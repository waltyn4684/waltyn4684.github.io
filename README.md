# Deployment and Local Hosting

This project is a responsive website built with HTML5 and CSS3.  
This README explains how I published the site with GitHub Pages and how I hosted it locally using XAMPP.

---

## GitHub Pages Hosting

**Live site:**  
https://waltyn4684.github.io/

### Steps I used
1. Opened the repo on GitHub.
2. Clicked **Settings**.
3. Clicked **Pages**.
4. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch  
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Saved and waited for the site to publish.

---

## Local Hosting Using XAMPP

### Tools
- XAMPP Control Panel (Apache)
- Web browser (Chrome / Edge)

### Steps I used
1. Installed and opened **XAMPP**.
2. Started **Apache**.
3. Copied my website project folder into the XAMPP `htdocs` folder.
4. Opened the site in the browser using:
   - `http://localhost/PROJECT-FOLDER/`
5. Confirmed the page loaded correctly.
